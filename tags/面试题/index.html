2015年最新前端开发面试题

目录 前言 面试需要注意几点 前端开发面试知识点大纲 必须掌握的知识点 HTML 部分 CSS 部分 JavaScript 部分 其他问题 前言本文总结了一些优质的前端面试题（多数源于网络），初学者阅后也要用心钻研其中的原理，重要知识需要系统学习，透彻学习，形成自己的知识链。万不可投机取巧，只求面试过关是错误的！ 面试需要注意几点 面试题目： 根据你的等级和职位变化，入门级到专家级：范围↑、深度↑、方向↑。 题目类型： 技术视野、项目细节、理论知识题，算法题，开放性题，案例题。 进行追问： 可以确保问到你开始不懂或面试官开始不懂为止，这样可以大大延展题目的区分度和深度，知道你的实际能力。 因为这种关联知识是长时期的学习，绝对不是临时记得住的。 回答问题再棒，面试官（可能是你的直接领导面试），会考虑我要不要这个人做我的同事？所以态度很重要。（感觉更像是相亲） 资深的工程师能把absolute和relative弄混，这样的人不要也罢，因为团队需要的你这个人具有可以依靠的才能（靠谱）。
